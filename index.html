You can use mouse over buttons or NumPad. If using NumPad, "Enter" is "=", exponent for scientific notation is lowercase "e". You can reset calculator by pressing lowercase "c".

This calculator is programming curricullum project. Use at your own risk. Also it is opensource - feel free to provide any corrections to calculations engine in JS code.
